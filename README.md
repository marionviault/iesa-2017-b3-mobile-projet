## How to work on the project

Install ionic :
```
$ sudo npm install -g cordova ionic
```

Install platforms :
```
$ ionic platform add ios
$ ionic platform add android
$ ionic build ios
$ ionic emulate ios
```

Launch iOS and android simulators :
```
$ ionic serve --lab
```
